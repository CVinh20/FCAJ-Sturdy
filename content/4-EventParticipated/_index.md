During my internship and learning journey with the First Cloud Journey community, I participated in three events related to prompt engineering, cloud computing, DevOps, cybersecurity, AI, and career orientation. Each event helped me gain practical knowledge, broaden my technical perspective, and better understand how modern cloud and software engineering practices are applied in real projects.

### [Event 1](4.1-Event1/)

&emsp;**Event Name:** Automated Prompt Engineering: Enhancing LLM Output Quality

&emsp;**Date & Time:** May 9, 2026

&emsp;**Location:** First Cloud Journey community event

&emsp;**Role:** Attendee

&emsp;**Summary:** This workshop focused on prompt engineering best practices, token economics, advanced prompting techniques, and the serverless architecture behind the Proptimizer browser extension. Through this event, I learned how structured prompts, clear context, role definition, and output constraints can improve LLM response quality and reduce unnecessary token usage.

&emsp;**Event Name:** First Cloud Journey Community Day

&emsp;**Date & Time:** June 6, 2026

&emsp;**Location:** First Cloud Journey community event

&emsp;**Role:** Attendee

&emsp;**Summary:** This community day included multiple sessions about Docker, AWS WAF, Machine Learning-based Network Intrusion Detection, AWS WebSocket architecture for Godot multiplayer, GraphRAG with Amazon Bedrock and Neptune, effective teamwork, and career growth from IT Helpdesk to System Administrator and Cloud/DevOps roles.
